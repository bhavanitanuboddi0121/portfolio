## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

The contact form falls back to the visitor's email client until Bhavani's EmailJS credentials are configured. The AI assistant requires `VITE_GROQ_API_KEY` as a local environment variable or GitHub Actions secret.
